Paket Pelatihan WebGIS

Pengantar/Deskripsi umum tentang tema pelatihan

Visualisasi data berbasis internet terus berkembang dengan pesat saat ini. Berbagai macam aplikasi pemetaan berbasis web juga terus bermunculan,beberapa di antaranya seperti google map, waze, lewatmana.com, fligthradar24.com dan lain sebagainya. Visualisasi data dipadukan dengan informasi lokasi telah menjadi trend untuk membantu dan menjawab berbagai kebutuhan masyarakat. Salah satu pemain besar dalam bidang web mapping adalah Google map.
Google map merupakan aplikasi peta dengan cakupan global dengan akurasi cukup mumpuni (dibandingkan dengan pesaingnya Microsoft, Yahoo atau Apple). Selain peta Jalan, Google Map juga menyediakan tampilan citra satelit dan terrain. Selain itu Google map juga menyediakan sebuah Application Program Interface (API).untuk para pengguna sehingga dapat menggunakan semua fasilitas pada aplikasi Google map sesuai dengan keinginan masing-masing. Bagaimana kita dapat membagun sebuah aplikasi sendiri dengan Google Map? Maka jawabannya tidak lain adalah dengan memahami bagaimana bekerja dan menggunakan Google Map API tersebut termasuk bahasa di baliknya yaitu Javascript.

Tujuan Pelatihan WebGIS

  1. Memahami dasar-dasar HTML dan Javascript.
  2. Memahami JSON dan Jquery.
  3. Dapat menampilkan peta Google.
  4. Dapat menambahkan data-data lain di atas peta Google.
  5. Dapat melakukan proses pencarian dan menampilkan objek yang dicari.
